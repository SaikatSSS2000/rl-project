# rl-project
Reinforcement Learn# CartPole Reinforcement Learning Project 🎮

This is a beginner-friendly Reinforcement Learning (RL) project where an AI agent learns to balance a pole on a moving cart using the PPO (Proximal Policy Optimization) algorithm.

---

## 🧠 Concept

The goal of this project is to demonstrate how an agent can learn from interaction with an environment using rewards.

Core idea:

State → Action → Reward → Learning

---

## 🎯 Environment

- Environment: CartPole-v1
- Library: Gymnasium
- Goal: Keep the pole balanced as long as possible

---

## ⚙️ Technologies Used

- Python
- Gymnasium
- Stable-Baselines3
